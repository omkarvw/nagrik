# Jan Aushadhi Medicine Savings Portal

## Vision
A civic-focused web application that helps Indian citizens find affordable generic medicine alternatives through the Pradhan Mantri Jan Aushadhi scheme. The app enables users to compare branded medicines with their Jan Aushadhi generic equivalents, discover potential savings, and locate nearby Jan Aushadhi stores.

## Problem Statement
Millions of Indians overpay for branded medicines when identical generic alternatives are available at 50-90% lower prices through government-approved Jan Aushadhi stores. Citizens lack:
- Awareness of generic alternatives for branded medicines
- Knowledge of actual price differences and savings potential
- Easy way to find nearby Jan Aushadhi stores
- Ability to share savings information with others

## Solution
An intuitive web application featuring:
- Medicine search with live autocomplete against generic and branded names
- Side-by-side comparison of branded vs Jan Aushadhi generic medicines
- Real-time savings calculation with visual emphasis
- Interactive map showing nearest Jan Aushadhi stores
- Shareable savings cards for social proof
- Fully offline-capable after first load

## Target Audience
- Indian citizens looking to reduce healthcare costs
- Families managing chronic medication expenses
- Healthcare workers advising patients on affordable options
- Senior citizens on fixed incomes

## Success Metrics
- Comprehensive medicine database coverage
- Accurate store location data
- Responsive design across all device sizes
- Offline functionality
- Sub-3 second initial load time

## Technical Stack
- Frontend: React 18 + TypeScript
- Build Tool: Vite
- Styling: Tailwind CSS
- Maps: Leaflet.js
- Icons: Material Symbols
- Font: Plus Jakarta Sans
- Data: Static JSON files in /public/data/

## Data Sources
1. **Product List**: janaushadhi.gov.in/product-list
2. **Store Locator**: Jan Aushadhi store locator API or scraped data

## Design System
- Warm civic palette with saffron (#FF9933), green (#056e00), cream backgrounds (#fff8f5)
- Plus Jakarta Sans typography
- Rounded shapes (12-16px border radius)
- Soft shadows for elevation
- Material Design icons
